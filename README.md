# 📣 Segmentação de Clientes RFM

> Segmentação avançada de clientes com análise RFM e clustering ML. Identifica campeões, em risco e oportunidades para campanhas personalizadas.

**🌐 Demo ao vivo (web app interativo):** https://jhondados.github.io/segmentacao-rfm/

[![Vertical](https://img.shields.io/badge/Vertical-Marketing-blue)]() [![IA](https://img.shields.io/badge/IA-Machine%20Learning-green)]() [![Demo](https://img.shields.io/badge/Demo-Interativa-orange)]()

## 📊 Resultados-chave (KPIs)
**8** Segmentos · **2.4M** Clientes · **+34%** Conversão · **5.2x** ROI Camp.

## 🧠 Tecnologias de IA
Machine Learning · Clustering · RFM

## ✨ O que faz
- Score RFM (Recência/Frequência/Valor)
- Clustering K-means + DBSCAN
- 8 personas acionáveis
- Recomendação de campanha por segmento

## 🗄️ Fonte de dados real
[E-commerce + CRM](https://www.gov.br/mdic)

## ⚙️ Como funciona
1. **Coleta de dados reais** — integração com E-commerce + CRM
2. **Processamento com IA** — Machine Learning · Clustering · RFM
3. **Dashboard & insights** — KPIs, gráficos e recomendações acionáveis
4. **Ação automatizada** — alertas, predições e decisões em tempo real
